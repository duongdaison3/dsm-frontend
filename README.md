# 💻 TOTAL DSM WebApp - Frontend Client

Giao diện người dùng cho nền tảng **Daily Standup Meeting (DSM)**. Được xây dựng theo phong cách SaaS hiện đại, tập trung vào trải nghiệm người dùng (UX) mượt mà và giao diện (UI) tối giản, chuyên nghiệp.

## 🛠️ Công nghệ sử dụng

- **Framework:** [Next.js 14](https://nextjs.org/) (App Router).
- **UI Library:** [React](https://react.dev/).
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework.
- **Language:** TypeScript.
- **Deployment:** [Vercel](https://vercel.com/).

## ✨ Điểm nổi bật về UI/UX

- **Thiết kế Responsive:** Hoạt động hoàn hảo trên mọi kích thước màn hình.
- **Auto-resize Textarea:** Form nhập liệu thông minh, tự động co giãn theo lượng text người dùng nhập.
- **Admin Dashboard:** Trực quan hóa dữ liệu bằng Progress Bar sinh động, hiển thị tỷ lệ nộp báo cáo theo thời gian thực.
- **SaaS Footer:** Tích hợp bộ đếm số lượng Standup toàn hệ thống gọi trực tiếp từ API.

## 🚀 Hướng dẫn cài đặt

**1. Clone kho lưu trữ này về máy:**

```
git clone [https://github.com/duongdaison3/dsm-frontend.git]
cd dsm-frontend
```

**2. Cài đặt các gói thư viện Node.js:**
`npm install`

**3. Khởi chạy Development Server:**
`npm run dev`

Mở trình duyệt (ví dụ: Edge) và truy cập `http://localhost:3000` để xem kết quả.
